# 🎨 그림 챌린지 커뮤니티 (Drawing Challenge Community)

> 누구나 가볍게 그리고, 함께 즐기는 창작 커뮤니티 — 브라우저에서 즉석으로 그리고, 주제 기반 챌린지로 참여하는 드로잉 서비스

연성대학교 Spring Boot 팀 과제 · 2026.06 (5인 팀 프로젝트)

---

## 기획 배경

Instagram·Pinterest 같은 기존 그림 커뮤니티는 완성도 높은 작품 위주라 초보자가 참여하기 부담스럽고, Procreate 같은 별도 드로잉 프로그램이 필요한 경우가 많습니다.

**그림 챌린지 커뮤니티**는 브라우저 Canvas에서 즉석으로 그리고, 매주 주어지는 챌린지 주제에 가볍게 참여하는 방식을 목표로 했습니다.

| | 기존 서비스 | 그림 챌린지 커뮤니티 |
|---|---|---|
| 드로잉 방식 | 별도 앱·프로그램 필요 | 브라우저 Canvas 직접 드로잉 |
| 참여 방식 | 자유 업로드 | 챌린지 주제 기반 참여 |
| 경쟁 요소 | 좋아요 수 표시만 | 랭킹 + 금/은/동 메달 수여 |
| 진입 장벽 | 높음 (완성도 중심) | 낮음 (참여 중심) |

---

## 핵심 기능

- 🖌 **브라우저 Canvas 드로잉** — 펜·지우개·채우기·스포이드 도구, 별도 프로그램 불필요
- ⏱ **주제 챌린지 + 5분 제한 시간** — 매주 새로운 주제 등록
- 🏆 **좋아요 랭킹 & 메달** — 챌린지 종료 시 1~3위에 금/은/동 메달 자동 부여
- 💬 **댓글 소통** — 다른 참여자 작품에 피드백
- 🗂 **진행중 / 종료 아카이브** — 종료된 챌린지 과거 작품 다시 열람
- 👤 **마이페이지** — 이메일 인증 회원가입, 닉네임·비밀번호 변경, 회원탈퇴

## 회원 기능 & 인증 구조

- **이메일 인증 회원가입**: Brevo SMTP로 인증 메일 발송 → 6자리 코드 검증 → BCrypt로 비밀번호 암호화 후 가입 완료
- **세션 기반 로그인**: Spring Session + Redis로 세션 관리. 로그인 시 HttpSession에 유저 정보 저장, JSESSIONID 쿠키로 인증 상태 유지
- **마이페이지**: 닉네임·비밀번호 변경, 회원탈퇴(세션 무효화) 지원. `/mypage` API로 로그인 유저 정보 조회

## 기술 스택

### Frontend
- React + TypeScript
- Vite
- Canvas API (드로잉)
- Axios

### Backend
- Spring Boot 3.3.5
- Spring Data JPA
- Spring Session + Redis (세션 인증)
- BCrypt (비밀번호 암호화)

### Infra (AWS)
- EC2 + nginx (배포 · 서빙)
- RDS MySQL (데이터 저장)
- Redis (세션 스토어)
- Brevo SMTP (이메일 인증)
